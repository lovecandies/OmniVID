# Export 模块蓝图

## 目标

将当前视频的字幕和结构化总结作为事实来源，优先调用已启用的 DeepSeek 扩写为详细文档，再渲染为 Markdown、DOCX 或 PPTX。

## 边界

- 导出内容必须以当前视频字幕和所选总结类型为依据，不编造视频外事实。
- DeepSeek 不可用或返回格式异常时，使用现有总结与字幕生成可下载的本地兜底文档。
- 三种文件格式共用 `ExportDocument`，避免内容口径不一致。
- 文件按请求即时生成，不新增导出任务表和文件持久化。

## 黑盒验收

1. 选择会议纪要并下载 Markdown -> 文件包含标题、执行摘要、详细章节、行动项和来源片段。
2. 下载 DOCX -> Office/Apache POI 可以打开且段落结构完整。
3. 下载 PPTX -> Office/Apache POI 可以打开且包含封面、摘要和内容页。
4. DeepSeek 不可用 -> 三种格式仍可下载，并通过响应头标记 `local-fallback`。
